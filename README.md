# Olympic Games Performance Analysis Dashboard

## Project Overview

This project presents an interactive **Power BI dashboard** built to analyze historical Olympic Games data. The dashboard provides insights into medal distribution, athlete participation, country performance, and sports trends across different Olympic events.

## Key Features

* KPI cards showing total medals, athletes, participating countries, sports, and Olympic Games
* Medal distribution analysis by country, gender, and medal type
* Trend analysis of medals across different Olympic years
* Treemap visualization for medal contribution by sport
* Interactive **year slicer** for dynamic data exploration
* Geographic map showing global participation of countries

## Tools & Technologies

* Power BI
* Data Visualization
* Data Cleaning & Transformation

## Insights

The dashboard helps identify top-performing countries, medal trends over time, and the contribution of different sports and genders to Olympic achievements.
